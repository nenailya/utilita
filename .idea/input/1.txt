Импорт рабочей области Eclipse
При работе с проектами Eclipse можно использовать мастер импорта проектов Eclipse
для создания проектов NetBeans из проектов рабочей области Eclipse и импорта путей к классам и других
параметров проекта. При использовании мастера импорта проектов Eclipse создание и настройка проекта NetBeans в
ручную не требуется. Откройте мастер импорта, выбрав "Файл" > "Импортировать проект" > "Проект Eclipse".
Дополнительные сведения о принципах работы мастера и возможностях одновременной работы с проектами
Eclipse и NetBeans см. в разделе Импорт проектов Eclipse в IDE NetBeans.

Настройка проекта Java на основе существующих исходных файлов
Для других проектов Java, разработанных вне среды NetBeans, рекомендуется использовать шаблон
"Существующие исходники" в мастере создания проектов для построения проекта NetBeans.
В мастере необходимо указать путь к исходным файлам и определить папку для метаданных проекта NetBeans.
Для настройки проекта используйте диалоговое окно "Свойства проекта".
